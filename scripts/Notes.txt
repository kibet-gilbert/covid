Creating a Singularity image from the Dockerfile: STEPS
1. Change directory to working directory:
$ cd /Path-to-install_directory/scripts
2. git clone the pangolin source code:
$ git clone https://github.com/cov-lineages/pangolin.git
$ cd pangolin
4. Build docker image from ./Dockerfile
$ sudo docker build -t local/pangolin.dif .
5. Search for docker image ID stored in /var/lib/docker:
$ sudo docker images
6. Create a tarball of the Docker image:
$ sudo docker save 48278907bf5c -o pangolin.tar
7. Convert the tarball to a singularity image:
$ sudo singularity build pangolin.sif docker-archive://pangolin.tar
